### Добавлено

#### Улучшенная система Bootstrapper

**Автоматическое создание ярлыка:**
- Bootstrapper теперь автоматически создаёт ярлык на рабочем столе при установке
- Ярлык создаётся с правильным описанием и рабочей директорией

**Документация:**
- Добавлен подробный README для системы bootstrapper
- Инструкции по настройке сервера для автообновлений
- Описание структуры файлов и процесса обновления

**Улучшенная конфигурация:**
- Добавлена опция `deleteAppDataOnUninstall: false` для сохранения настроек пользователя
- Обновлены описания в package.json bootstrapper
